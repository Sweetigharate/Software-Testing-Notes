Scratch — Core Concepts

What is Software Testing?

Software testing is the process of evaluating a system to find defects and verify it meets requirements. The goal is to deliver quality software that works correctly for real users.

Verification — are we building it right? Validation — are we building the right thing? Quality Assurance Bug Detection

Black Box / White Box / Grey Box

Black Box Testing

No knowledge of internal code. Tests only inputs and outputs.
Example: Enter SKU, verify product appears.

White Box Testing

Full knowledge of code. Tests internal logic, branches, paths.
Example: Dev tests every if/else in payment code.

Grey Box Testing

Partial knowledge — knows APIs, DB schema, not full code.
Example: QA knows API contract, tests full flow.

Pro — Test Design Techniques

Equivalence Partitioning

Divide inputs into valid/invalid partitions. Test one value per partition. Example: Pin code field (6 digits only)

Valid: 400001 Invalid: 40000 (5 digits) Invalid: 4000011 (7 digits) Invalid: ABC123 (letters)

Boundary Value Analysis

Test at exact boundaries and just beyond. Age field: 18–60 allowed.

17 — below (fail) 18 — lower boundary (pass) 60 — upper boundary (pass) 61 — above (fail)

Page Object Model

POM — Separates locators from test logic

pages/LoginPage.js
class LoginPage {
  constructor(page) {
    this.mobileInput = page.locator('input[name="mobileNumber"]');
    this.otpInput    = page.locator('input[name="otp"]');
    this.submitBtn   = page.locator('button[type="submit"]');
  }
  async login(mobile, otp) {
    await this.mobileInput.fill(mobile);
    await this.submitBtn.click();
    await this.otpInput.fill(otp);
    await this.submitBtn.click();
  }
}
// In test file — clean, no raw selectors
await new LoginPage(page).login('9876543210', '1234');

Data-Driven Testing

const products = [
  { sku: 'JM-001', name: 'JioFi Router' },
  { sku: 'JM-002', name: 'Lloyd AC' },
];
for (const { sku, name } of products) {
  test(`Search: ${name}`, async ({ page }) => {
    await page.fill('#search', sku);
    await expect(page).toContainText(name);
  });
}

Waits

Three Types of Waits

Implicit Wait — avoid
// Applies globally — masks real issues
driver.manage().timeouts()
  .implicitlyWait(Duration.ofSeconds(10));
Explicit Wait — use this
WebDriverWait wait =
  new WebDriverWait(driver,
    Duration.ofSeconds(15));
wait.until(
  ExpectedConditions
    .elementToBeClickable(
      By.id("add-to-cart")));
Full Example — Python

Login + Search + Add to Cart

from selenium import webdriver
from selenium.webdriver.common.by import By
from selenium.webdriver.support.ui import WebDriverWait
from selenium.webdriver.support import expected_conditions as EC

driver = webdriver.Chrome()
wait  = WebDriverWait(driver, 15)
try:
    driver.get("https://www.jiomart.com")
    wait.until(EC.presence_of_element_located(
      (By.NAME, "mobileNumber"))).send_keys("9876543210")
    driver.find_element(By.XPATH,
      "//button[@type='submit']").click()
    wait.until(EC.visibility_of_element_located(
      (By.NAME, "otp"))).send_keys("1234")
    wait.until(EC.element_to_be_clickable(
      (By.CSS_SELECTOR, ".product-card:first-child"))).click()
    wait.until(EC.element_to_be_clickable(
      (By.ID, "add-to-cart-btn"))).click()
    print("Product added to cart")
except Exception as e:
    print(f"Failed: {e}")
finally:
    driver.quit()

Locators

Best Practice Locators

Use these first
  • getByRole('button', {name:'Submit'})
  • getByLabel('Mobile Number')
  • getByPlaceholder('Search...')
  • getByTestId('cart-icon')
  • getByText('Add to Cart')
Use carefully
  • locator('.add-cart-btn')
  • locator('#submit')
  • locator('//button')
  • locator('nth=0')
Full E2E — JioMart COD Order

Complete Order Placement Test

import { test, expect } from '@playwright/test';

test('Place COD order', async ({ page }) => {
  // Login with hardcoded OTP (SIT env)
  await page.goto('https://sit.jiomart.com');
  await page.getByPlaceholder('Enter mobile number').fill('9876543210');
  await page.getByRole('button',{name:'Get OTP'}).click();
  await page.getByPlaceholder('Enter OTP').fill('1234');
  await page.getByRole('button',{name:'Verify'}).click();
  await expect(page.getByTestId('user-avatar')).toBeVisible();

  // Search SKU
  await page.getByRole('searchbox').fill('JM-9876');
  await page.keyboard.press('Enter');

  // Click product → Add to Cart
  await page.locator('.product-item').first().click();
  await page.getByRole('button',{name:'Add to Cart'}).click();
  await expect(page.getByTestId('cart-count')).toHaveText('1');

  // COD Checkout
  await page.goto('/checkout');
  await page.getByLabel('Cash on Delivery').check();
  await page.getByRole('button',{name:'Place Order'}).click();
  await expect(page.getByText('Order Placed Successfully')).toBeVisible();

  // Capture Order ID for SAP CRM verification
  const orderId = await page.getByTestId('order-id').textContent();
  console.log(`Order ID: ${orderId}`);
});
